TIS的`v4.3.0`版本终于完成了开发，`v4.3.0`在以下几方面进行了功能加强和升级。https://github.com/datavane/tis/milestone/9?closed=1

1. 增加Apache Paimon数据端的批量与实时写入能力
2. 第三方开源依赖包版本升级
    * Flink版本升级到`v1.20.1`
    * Flink-CDC版本升级到`v3.4.0`
3. 实时同步管道增加执行状态可视化功能项，提供提供实时任务限流&暂停&泄洪功能可以应对数据流上游突发需求，例如：DBA夜晚大批量订正业务数据，导致大量数据堆积，TIS的实时同步管道会进行限流，防止数据积压，从而保证数据正常同步。
4. 增加 新人引导功能，当第一安装使用TIS的用户进入TIS控制台，可以用过该指南快速地熟悉TIS操作流程。
